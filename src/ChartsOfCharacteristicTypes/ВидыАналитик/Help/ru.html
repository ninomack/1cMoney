Аналитика статей

Аналитика статей предназначена для организации более подробного и точного учета доходов и расходов. С помощью аналитики для каждой статьи дохода или расхода задаются дополнительные параметры учета, что позволяет рассматривать одну и ту же статью с разных точек зрения.

Например, с помощью аналитики "Члены семьи", включенной для статей доходов, можно получить определить, кто больше доходов приносит в семью по статье "Зарплата", а кто - по статье "Сторонние заказы"?
С помощью той же аналитики "Члены семьи", включенной для статей расходов, можно определить, на кого из членов семьи больше тратится денег по группе статей "Здоровье и внешний вид"? А по группе статей "Развлечения"?

Использование аналитики статей включается отдельно для статей доходов и для статей расходов. Расположение аналитики в форме операций можно настроить с помощью кнопки Аналитика.
