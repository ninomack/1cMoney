Настройка заполнения аналитики в формах операции

В меню каждой формы операции, в которой допускается использовать аналитику статей, есть команда , которая позволяет настроить расположение аналитики в форме операции.

В диалоге Настройка заполнения аналитики есть два списка, в которых перечислена вся доступная для операции аналитика. С помощью стрелок вверх и вниз аналитику можно перемещять из нижнего списка в верхний, и наоборот, задавая тем самым способ ее заполнения: в шапке операции (верхний список) или в таблице (нижний список).

Если значение аналитики одинаково для всех статей операции, то такую аналитику удобно заполнять в шапке (разместить в верхнем списке настройки). Пример: аналитика "Получатель оплаты" для покупки в магазине имеет одно значение для всех статей расходов, ведь покупка осуществляется в одном магазине.

Если для каждой статьи требуется указать разные значения аналитики, то сделать это можно только в строках таблицы. Пример: расходы по статьям "Сладости" и "Спиртные напитки" могут быть сделаны по разным членам семьи, потому аналитику "Члены семьи" размещают в колонках таблицы (в нижнем списке настроек).

Программа запоминает сделанные настройки для каждой операции и каждого шаблона, по этому операции будут всегда выглядеть так, как их сохранили.
