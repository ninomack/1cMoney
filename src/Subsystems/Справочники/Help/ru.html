Справочники

Справочники предназначены для хранения информации об используемых в учете объектах: кошельках, контактах, долгах и т.д.

Записи справочников можно просматривать, редактировать, помечать на удаление или изменять их актуальность, если это не запрещено для конкретного пользователя. Основные справочники перечислены в панели команд, а с помощью главного меню программы Операции — Справочник… можно открыть список всех справочников, доступных пользователю.

В операциях и других формах элементы справочников указываются с помощью выбора из списка. Например, в операции Перемещение установите курсор в поле Из кошелька. Программа предложит список последних использовавшихся в этой операции кошельков. Если нужного кошелька в этом списке не оказалось, нажмите на ссылку Показать все или клавишу F4 для открытия полного списка элементов справочника и выберите необходимые данные с помощью двойного щелчка мышью или клавиши Enter.

Также, ввод данных из справочников возможен вручную. Для этого следует в поле ввода ввести с клавиатуры несколько букв (символов) наименования - если в справочнике есть данные, содержащие введенные символы, они будут предложены для выбора в выпадающем списке. Если элемент с нужным наименованием не найден, его можно создать прямо из поля, в котором набираете его наименование, нажав на ссылку Создать или клавишу F8.
