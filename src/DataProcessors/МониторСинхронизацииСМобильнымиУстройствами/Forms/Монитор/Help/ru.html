Синхронизация с мобильными устройствами

Форма предназначена для управления синхронизацией и просмотра результатов обмена с мобильными устройствами

 

В верхней части формы находится информация об используемых способах обмена и команды настройки этих способов и всей синхронизации в целом.

 

 

В первой колонке настроек отображаются параметры облачного сервиса: имя провайдера, логин пользователя и команда выполнения синхронизации. Открыть настройки облачного сервиса можно нажатием на «шестеренку» или по ссылке с логином пользователя. Если сервис не настроен, вместо логина пользователя будет указано состояние сервиса, например «выключен».
С помощью ссылки Синхронизировать выполняется немедленный обмен файлами с заданным облачным сервисом. По ссылке Как использовать? можно получить дополнительные сведения о настройках и использовании облачного сервиса.

 

В средней колонке настроек отображается информация о настройках домашней сети. Сами настройки можно открыть нажатием на «шестеренку» или по ссылке с адресом информационной базы. Если домашняя сеть не используется или адрес не задан, вместо адреса будет отображаться текущее состояние настроек, например, слово «Выключена».
С помощью ссылки Проверить можно проверить доступность заданного в настройках адреса.

Обратите внимание: указание адреса в настройках не обязательно, он нужен только для удобства и проверки.
База может быть опубликована в конфигураторе и будет доступна для синхронизации независимо от указания ее адреса в настройках домашней сети.

 

В третьей колонке находится меню управления синхронизацией:

  • Обновить (F5) – обновляет информацию всей формы
  • Загрузить данные из файла – используется для чтения данных из файла, скопированного с мобильного устройства, и выгрузки изменений в файл ответа
  • Включить (отключить) обмен через облачный сервис – используется для включения или отключения способа обмена через облачный сервис. Например, можно временно отключить этот способ обмена для выполнения профилактических работ в сервисе
  • Включить (отключить) обмен в домашней сети – используется для включения или отключения способа обмена в домашней сети. Например, можно временно отключить этот способ обмена для обновления ПО или выполнения профилактических работ
  • Временно отключить всю синхронизацию – устанавливает запрет на использование любого способа обмена с этой базой. Например, при выполнении регламентных работ с базой.
  • Как работает синхронизация – открывает общее описание синхронизации с мобильными устройствами

  

 

В списке ниже анели настроек выводятся все мобильные устройства, которые когда-либо были синхронизированы с этой базой.

·         Колонка Мобильное устройство/Пользователь показывает наименование мобильного устройства и его пользователя. Нажатием Enter или двойным кликом мыши по этой колонке открывается форма мобильного устройства.

·         Колонка Получено с устройства отображает результаты последней загрузки данных, полученных с мобильного устройства. В верхней части строки выводится результат загрузки данных в эту базу данных. В нижней части строки – справочные сведения о последней выгрузке данных с мобильного устройства. Таким образом, видно, когда были отправлены данные, загруженные в эту базу. Клик по ссылкам в этой колонке открывает форму подробного описания результата загрузки.

·         Колонка Отправлено на устройство отображает результаты отправки данных из этой базы на мобильное устройство. В верхней части строки – результат выгрузки из этой базы, в нижней – справочная информация о последней загрузке данных на устройстве.

      Обратите внимание: при нормальном обмене через облачный сервис или через файлы дата последней загрузки на устройстве будет всегда меньше даты последней отправки данных на это устройство.

Клик по ссылкам в этой колонке открывает форму подробного описания результата отправки данных.

 

В нижней части формы продублирована ссылка на команду загрузки файла, скопированного с мобильного устройства. Эта возможность бывает необходима в случаях, когда другие способы синхронизации недоступны, например, при отсутствии интернета и wi-fi.

 

Ответы на некоторые вопросы по работе с формой

  • Как посмотреть список объектов, полученных с мобильного устройства?
    В строке нужного устройства кликните по любой ссылке в колонке «Получено с устройств». Откроется подробное описание результата, там будет ссылка «Получено с устройства»
  • Как посмотреть/изменить список объектов, помеченных для отправки на устройство?
    В строке нужного устройства кликните по любой ссылке в колонке «Отправлено на устройства». Откроется подробное описание результата, в котором будет ссылка «Зарегистрировано к отправке»
  • Как узнать причину ошибки?
    В строке устройства кликнуть по красной ссылке с текстом об ошибке. Откроется подробное описание результата синхронизации, в котором будет и описание ошибки
  • Как добавить в список новое устройство?
    Устройство добавляется автоматически при первой его синхронизации с настольной программой. Важно, чтобы приложение на устройстве не было синхронизировано ни с какой другой базой данных, иначе синхронизация не получится.
    Задайте на устройстве настройки синхронизации и выполните первый обмен. Настольная программа сама добавит в список новое устройство.
  • Как запретить синхронизацию?
    Для запрета синхронизации всех устройств сразу достаточно выключить синхронизацию с помощью команды меню Управление… –> Временно отключить всю синхронизацию. Например, это бывает полезно при выполнении каких-либо регламентных работ на компьютере.
    Для запрета синхронизации отдельного устройства достаточно открыть его карточку и снять флаг Обмен разрешен.
    Если устройство больше не будет использоваться, его можно пометить на удаление кнопкой Del или соответствующей командой меню.
