Форма имущества (создание нового)

Имуществом может быть любая ценная для пользователя вещь. Желательно давать имуществу такое Наименование, которое позволит быстро находить его в списке и не путать с другим имуществом. Для удобства, имущество можно поместить в какую-нибудь Папку (группу) (например: "Автотранспорт", "Недвижимость", "Бытовая техника"). Группу можно создать непосредственно из формы имущества кнопкой

В момент создания имущества можно указать его начальную стоимость. Стоимость каждого объекта имущества учитывается только в одной Валюте, которая задается при создании имущества и не может быть изменена в дальнейшем.

Если в форме создания имущества заполнены реквизиты Начальная стоимость и На дату, то одновременно с созданием имущества программа запишет новую операцию Ввод / корректировка остатков на указанные дату и сумму. В последующем эту операцию можно легко изменить или удалить.
