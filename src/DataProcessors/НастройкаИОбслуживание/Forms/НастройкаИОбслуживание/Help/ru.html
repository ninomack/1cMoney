Настройка и обслуживание

Обработка содержит все основные настройки программы и команды по ее обслуживанию, администрированию и обновлению.
Для быстрого поиска нужной команды или настройки используйте строку поиска в верхней части формы.

Настройки программы
содержат настройки, изменение которых затронет всех пользователей программы

Настройки пользователя
содержит индивидуальные настройки кажого пользователя. При необъодимости такие настройки могут быть скопированы между пользователями.

Резервное копирование
содержит настройки резервного копирования и команды создания резервной копии и восстановления данных из резервной копии.

Администрирование
содержит команды по управлению фоновыми заданиями, активными пользователями и журналом регистрации событий.

Управление пользователями
содержит команды по управлению списком пользователей, их доступом к программе и настройками

Регламентные операции
содержит команды, помогающие поддерживать порядок в информационной базе: удалять лишние объекты, пересчитывать остатки, обновлять индекс для поиска и т.п.

Обновление программы
содержит все необходимое для регистрации в интернет-поддержке и автоматического или ручного обновления программы

Адресный классификатор
позволяет настроить адресный классификатор для заполнения декларации 3-НДФЛ
