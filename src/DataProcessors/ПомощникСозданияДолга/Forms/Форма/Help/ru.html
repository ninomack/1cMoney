Создание нового долга

С помощью долга можно описать выданный или полученный заем (кредит) и задать параметры его погашения.

Помощник создания долга автоматизирует ввод соответствующих операций, при необходимости создает шаблоны и добавляет записи в расписание.

Шаг 1, Оснонвые сведения о долге:

Контакт - контакт, с которым возникли долговые отношения. С каждым контактом можно учитывать несколько долгов: несколько кредитов в одном банке, несколько выполненных заказов и т.д.

Вид долга - определяет правила учета остатков и ведения взаиморасчетов. Выберите подходящий вариант:

  • Нам должны - вы выдали взаймы деньги или выполнили в долг работы. Остаток долга учитывается в разделе "Нам должны", отрицательный остаток расценивается как ошибка.
  • Мы должны - вы получили в долг деньги, взяли кредит или обязуетесь оплатить чьи-то работы/услуги. Остаток долга учитывается в разделе "Мы должны", положительный остаток расценивается как ошибка.
  • Взаимный расчет - вы с контактом многократно кредитуете друг-друга и определить кто кому должен можно только по конечному остатку. Такой долг отображается одновременно в разделах "Нам должны" и "Мы должны", но остаток учитывается только в одном из разделов, в зависимости от его знака.

Валюта - валюта, в которой учитывается остаток задолженности. Валюта задается при создании долга и ее невозможно изменить, если по долгу уже начался учет. При этом выдача и погашение долга может осуществляться в любых валютах.

Наименование долга - произвольное название долга, которое позволит отличить его от других долгов. Можно воспользоваться автоматическим наименованием, выбрав его из списка этого поля.

Комментарий - любая дополнительная информация.

Начальный остаток - в этой группе можно указать сумму и дату начального остатка. Заполняйте эти поля только в том случае, если на начало ведения учета в программе по добавляемому долгу действительно был остаток.

Для указания сроков возврата долга нажмите кнопку Далее. Для записи долга в информационной базе нажмите кнопку Записать и закрыть.

 

Шаг 2, Условия погашения долга:

Задать срок / график погашения - установите этот флажок, если хотите указать срок погашения кредита или заполнить график погашения, например, графиком, полученным из банка. После установки этого флажка станут доступны следующие поля:

Способ погашения - определяет способ, которым будет погашаться долг. Доступны следующие варианты:

  • Разовый платеж - используется для погашения всей суммы долга за один раз.
  • Аннуитетные платежи и Дифферинцированные платежи - используются для соответствующих кредитных договоров с банками.
  • Произвольные платежи - используются для указания произвольных сумм и сроков погашения долга. Например, погашать в чечение месяца по 2000 руб. каждый понедельник.

Задать проценты и комиссию - установите этот флажок, если хотите задать процентную ставку и / или комиссию, связанную с обслуживанием этого долга. После установки этого флажка станут доступны следующие поля:

Процентная ставка - ставка процентов за год.

Статья учета процентов - статья расхода для полученного долга или статья доходов для выданного долга, по которой будет учитываться сумма процентов.

Ежемесячная комиссия - сумма взыскиваемой ежемесячно комисии.

Статья учета комиссии - статья расхода для полученного долга или статья доходов для выданного долга, по которой будет учитываться сумма комиссии.

 

В зависимости от указанного способа погашения программа предложит заполнить разные дополнительные поля. Для разового платежа требуется заполнть:

Из кошелька (в случае выданного долга - В кошелек) - кошелек, из которого планируется погашать долг (или в который ожидается получить возвращаемую сумму).

Когда вернем (в случае выданного долга - Когда вернут) - дата возврата долга

Сумма - сумма, ожиадемая к возврату.

После заполнения указанных реквизитов долг можно сохранить в информационной базе нажмите кнопкой Записать и закрыть. Для возврата к предыдущему шагу используйте кнопку Назад.

 

Для способа погашения Аннуитетные платежи и Дифферинцированные платежи требуется заполнить следующие реквизиты:

Сумма долга - сумма, указанная в договоре с банком.

Начало срока - дата, с которой начался отсчет срока кредита

Срок (месяцев) - срок, как он указан в договоре.

Количество дней в году - способ расчета количества дней, как он указан в банковском договоре или приложении к нему.

Платить каждое ... числи месяца - как указано в договоре с банком.

 Для заполнения графика платежей нажмите кнопку Далее. Для записи долга в информационной базе нажмите кнопку Записать и закрыть.

 

Шаг 3, График погашения долга.

Для способа погашения Аннуитетные платежи и Дифферинцированные платежи график можно рассчитать автоматически с помощью кнопки Пересчитать график. Так же, при изменении сумм в какой-либо строке графика все последующие строки графика пересчитываются автоматически. Вы можете отключить этот пересчет с помощью флажка При редактировании не пересчитывать основной долг и проценты.

Приведите график погашения в соответствие с графиком, полученным в банке. После этого долг можно сохранить в информационной базе нажмите кнопкой Записать и закрыть. Для возврата к предыдущему шагу используйте кнопку Назад.

 

Одновременно с записью нового долга программа автоматически создаст операцию Ввод / изменение остатка на сумму начального долга. Если задана дата погашения или расписание, помощник создаст соответствующий шаблон операции с графиком погашения.
