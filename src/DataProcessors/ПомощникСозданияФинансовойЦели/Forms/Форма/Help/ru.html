Создание Финансовой цели

С помощью финансовой цели можно зарезервировать сумму накоплений в пределах одного и того же кошелька. Например, если часть денег на банковском депозите зарезервировать под финансовую цель "Новая квартира", то эта сумма не будет использоваться в распределении накоплений на другие цели.

Помощник создания финансовой цели позволяет автоматически зарезервировать имеющиеся накопления на создаваемую цель.

Шаг 1, Параметры финансовой цели

Наименование - произвольное название финансовой цели, которое позволит отличить ее от других целей.

Сумма цели - предположительная сумма, достаточная для достижения цели. Это может быть, например, сумма пенсионных накоплений, стоимость дорогостоящей вещи, предельная сумма затрат на ремонт или другое мероприятие, и т.п. Сумму цели указывать не обязательно, накопления можно делать и без указания предельной суммы.

Валюта - валюта, в которую пересчитываются накопления на финансовую цель. Задается при создании финансовой цели и не меняется после начала учета операций по цели.

Срок достижения - дата, к которой планируется достичь финансовую цель. Срок указывать не обязательно, накопления можно делать и без ограничения сроком.

Шаблон накоплений, график накоплений - основной шаблон и расписание, с помощью которого планируются накопления на финансовую цель.

Комментарий - любая дополнительная информация.

Для резервирования свободных накоплений на создаваемую цель нажмите кнопку Далее. Для записи финансовой цели в информационной базе нажмите кнопку Записать и закрыть.

Шаг 2, Перенос доступных накоплений на финансовую цель

В таблице перечислены все кошельки для накоплений и для каждого указана сумма доступных накоплений - т.е. накоплений, сделанных без указания конкретной цели.

Укажите сумму, которую желаете зарезервировать на создаваемую цель, в колонке  Перенести в накопления на цель. Воспользуйтесь кнопкой Перенести все доступные накопления, что бы автоматически заполнить эту колонку по всем кошелькам.

Для записи финансовой цели в информационной базе нажмите кнопку Записать и закрыть. Для возврата на предыдущий шаг нажмите кнопку Назад.

Одновременно с заиписью финансовой цели программа создаст перемещения по кошелкам, для которых была заполнена колонка  Перенести в накопления на цель.
