Справочник "Долги"

Справочник Долги содержит информацию обо всех долговых отношениях, возникающих у пользователя с его контактами. С каждым контактом одновременно можно учитывать несколько долгов, например: несколько кредитов в одном и том же банке, несколько выполенных но неоплаченных заказчиком работ, несколько выданных коллеге взаймы сумм и т.д..

В программе могут учитываться следующие Виды долгов:

  • Выданный заем - предназначен для ведения расчетов с контактами, имеющими долг перед пользователем. Например, с помощью такого долга учитываются выданные взаймы деньги; долги заказчика за выполненные, но еще не оплаченные работы; долг арендатора за используемое имущество и т.д.
  • Полученный кредит - предназначен для ведения расчетов по долгам пользователя перед контактами. К таким долгам могут относиться банковские кредиты; суммы, полученные в долг от коллег и знакомых; начисленные, но еще не оплаченные коммунальные платежи и т.п.
  • Взаимные расчеты - предназначен для учета расчетов с контактами, которых невозможно однозначно отнести к кредиторам или заемщикам. Например, знакомый дает вам некоторую сумму до зарплаты, потом вы за него оплачиваете ужин в кафе, потом уже вы даете ему в долг до зарплаты... С помощью вида долга взаимные расчеты в любой момент можно определить: кто кому и сколько должен.

Для отслеживания долговых обязательств и сроков в программе используются шаблоны операций с расписаниями особого вида: графиками погашения долгов. Такие графики погашения можно создавать не только по банковским кредитам, но и по прочим полученным или выданным в долг деньгам. Например: пользователь может создать график ежемесячного возврата денег, взятых в долг у своих родственников.

Все запланированные в графике суммы учитываются в календаре и при составлении бюджета.

Команды списка справочника

Для открытия списка справочника используйте команду Долги в меню Справочники панели команд. Управлять содержимым справочника можно с помощью следующих команд: 

Обратите внимание: в зависимости от ширины формы часть команд может автоматически скрываться в меню Еще.
  • Создать открывает форму помощника создания нового долга.
  • Скопировать - создаст копию выделенного в списке элемента и откроет его форму для редактирования. 
  • Изменить - откроет форму элемента для изменения.
  • Изменить остатки - создаст и откроет операцию Ввод/изменение остатка для выделенного в списке элемента справочника.
  • Пометить на удаление / снять пометку
    • установит или снимет пометку на удаление. 
  • Обновить - обновит список справочника, покажет изменения, сделанные на другом компьютере или в другой сессии.
  • Найти - откроет диалог для поиска элементов справочника. При применении поиска в списке справочника останутся только те элементы, которые содержат искомую информацию.
  • Отменить поиск - отменяет отбор, наложенный на список командной Найти. 
  • Настроить список - открывает диалог, в котором можно задать сортировку, отбор и условное оформление списка справочника.
  • Выбрать настройки - открывает диалог выбора настроек, сохраненных пользователем ранее.
  • Сохранить настройки - открывает диалог для сохранения настроек списка под каким-либо именем, для быстрого примерения в дальнейшем.
  • Установить стандартные настройки - сбрасывает все изменения настроек, сделанные пользователем.
  • Вывести список - открывает диалог для вывода списка в табличный документ (отчет).
  • Изменить форму откроет диалог для изменения внешнего вида формы. Можно изменить расположение полей и надписей, изменить колонки и т.д.
  • Справка открывает данный текст справки.
