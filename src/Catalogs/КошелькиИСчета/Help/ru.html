Справочник "Кошельки и счета"

Справочник Кошельки и счета содержит все места хранения наличных и безналичных денег: кошельки, бумажники, банковские счета, пластиковые карты и пр. В каждом кошельке может учитываться только одна валюта, которая задается при создании кошелька и не может быть изменена в дальнейшем.

Кроме валюты, важным свойством кошелька является тип кошелька (счета):

  • Наличность - кошелек, предназначенный для хранения только наличных денег. Отрицательный остаток по такому кошельку расценивается программой как ошибка в учете и показывается красным цветом.
  • Банковская карта дебетовая - пластиковая карта, по которой не предусмотрен овердрафт или банковский кредит. Отрицательный остаток по такой карте расценивается программой как ошибка в учете (показывается красным).
  • Банковская карта кредитная - пластиковая карта, которую предполагается использовать только для покупок в кредит. Остаток долга по такой карте показывается в разделе Кредитные карты, а не в деньгах, тем самым общий остаток денег не зависит от долга по кредитной карте.
    Если по такой карте образуется положительный остаток, то он будет расцениваться программой как ошибка в учете и показывается красным цветом.
  • Банковская карта универсальная - пластиковая карта, по которой любой остаток (положительный или отрицательный) не считается ошибкой. Например: дебетовая карта с разрешенным овердрафтом. Положительный остаток по таким картам будет отображаться в разделе Деньги, а отрицательный - в разделе Кредитные карты.
  • С помощью типов Банковский счет, Электронный кошелек, Прочее можно указать другие места хранения денег. Отрицательные остатки по таким кошелькам программа расценивает как ошибку и показывает красным.

Некоторые кошельки могут быть предназначены для накоплений. Деньги в таких кошельках не используются на текущие расходы и оперативные платежи, а собираются в течение продолжительного времени для достижения одной или нескольких финансовых целей. Программа помогает контролировать целевое использование денег в таких кошельках.

Для удобства кошельки можно объединять в группы. Например: можно сделать такие группы:

  •  для всех счетов и пластиковых карт одного банка;
  •  для всех мест хранения наличных денег;
  •  для кошельков и счетов, предназначенных для накоплений

Тогда итоги по группам покажут общие остатки наличных денег, безналичных денег, накоплений и т.д.

Команды списка справочника

Для открытия списка справочника используйте команду Кошельки и счета в меню Справочники панели команд. Управлять содержимым справочника можно с помощью следующих команд:

Обратите внимание: в зависимости от ширины формы часть команд может автоматически скрываться в меню Еще.
  • Создать - открывает форму нового элемента справочника.
  • Создать группу - открывает форму новой группы.
  • Скопировать - создает копию выделенного в списке элемента и открывает его форму для редактирования.
  • Изменить - открывает форму элемента для изменения.
  • Долги - открывает список долгов выделенного контакта.
  • Пометить на удаление / снять пометку
    • - устанавливает или снимает пометку на удаление.
  • Обновить - обновляет список справочника для отображения изменений, сделанных на другом компьютере или в другой сессии.
  • Режим просмотра - открывает меню для изменения режима просмотра списка - в виде дерева, в виде сплошного списка всех групп и элементов, или в виде иерархического списка, в котором содержимое каждой группы просматривается отдельно от прочих групп и элементов.
  • Переместить в группу - открывает диалог выбора группы, в которую следует переместить выделенные в списке записи.
  • Перейти на уровень выше - выделяет в списке группу, в которую входит текущий элемент.
  • Перейти на уровень ниже - выделяет первый элемент, входящий в выделенную группу.
  • Найти - открывает диалог для поиска элементов справочника. При применении поиска в списке справочника останутся только те элементы, которые содержат искомую информацию.
  • Отменить поиск - отменяет отбор, наложенный на список командной Найти.
  • Настроить список - открывает диалог, в котором можно задать сортировку, отбор и условное оформление списка справочника.
  • Выбрать настройки - открывает диалог выбора настроек, сохраненных пользователем ранее.
  • Сохранить настройки - открывает диалог для сохранения настроек списка под каким-либо именем, для быстрого примерения в дальнейшем.
  • Установить стандартные настройки - сбрасывает все изменения настроек, сделанные пользователем.
  • Вывести список - открывает диалог для вывода списка в табличный документ (отчет).
  • Изменить форму - открывает диалог для изменения внешнего вида формы. Можно изменить расположение полей и надписей, изменить колонки и т.д.
  • Справка - открывает данный текст справки.
