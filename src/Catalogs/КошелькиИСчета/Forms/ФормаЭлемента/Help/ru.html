Форма справочника "Кошельки и счета"

"Кошельком" в программе называется любое место хранения денег: бумажники, конверты, банковские счета и банковские карточки, виртуальные кошельки платежных систем и прочее.

Реквизиты справочника:

Наименование -  произвольное название кошелька, которое позволит отличить его от других кошельков.

Тип кошелька / счета - тип кошелька как хранилища денег. С помощью этого реквизита задаются правила учета остатков:

  • Наличность - кошелек, предназначенный для хранения только наличных денег. Отрицательный остаток по такому кошельку расценивается программой как ошибка в учете и показывается красным цветом.
  • Банковская карта дебетовая - пластиковая карта, по которой не предусмотрен овердрафт или банковский кредит. Отрицательный остаток по такой карте расценивается программой как ошибка в учете (показывается красным).
  • Банковская карта кредитная - пластиковая карта, которую предполагается использовать только для покупок в кредит. Остаток долга по такой карте показывается в разделе Кредитные карты, а не в деньгах, тем самым общий остаток денег не зависит от долга по кредитной карте.
    Если по такой карте образуется положительный остаток, то он будет расцениваться программой как ошибка в учете и показывается красным цветом.
  • Банковская карта универсальная - пластиковая карта, по которой любой остаток (положительный или отрицательный) не считается ошибкой. Например: дебетовая карта с разрешенным овердрафтом. Положительный остаток по таким картам будет отображаться в разделе Деньги, а отрицательный - в разделе Кредитные карты.
  • С помощью типов Банковский счет, Электронный кошелек, Прочее можно указать другие места хранения денег. Отрицательные остатки по таким кошелькам программа расценивает как ошибку и показывает красным.
  • Валюта - валюта денежных средств кошелька; задается при его создании. Валюту нельзя изменить, если по кошельку уже ведется учет.

Кредитный лимит - максимальная сумма кредита, разрешенная банком по кредитной карте, или максимальная сумма овердрафта для дебетовой карты. Используется только для банковских карт. 

Назначение кошелька - определяет, для каких целей будут использоваться деньги кошелька.

  • На текущие расходы - кошелек  используется для оплаты повседневных расходов, не требующих накопления денег. Именно по остаткам этих кошельков осуществляется основное планирование и прогнозирование. Обычно, таким кошелькам соответствуют бумажники, банковские карточки, электронные кошельки.
  • Для накоплений - кошелек используется для накопления денег, а при необходимости - для оплаты расходов на достижение финансовых целей. Остатки таких кошельков могут быть распределены по финансовым целям, а в планировании и прогнозировании они учитываются отдельно. Кроме того, в каждой операции, в которой используется кошелек для накоплений, программа требует указать финансовую цель, для которой используются деньги.

Входит в группу - группа кошельков. Можно объединить кошельки в группы для удобства работы и получения сводных остатков.

Комментарий - любая дополнительная информация.

Актуален - признак актуальности элемента справочника. Снимите этот флажок, чтобы элемент не отображался в списках.

Текущий остаток - в группе показана сумма текущего остатка денег в кошельке. Изменить остаток можно с помощью одноименной команды.

Команды формы справочника

В форме элемента справочника можно использовать следующие команды:

Обратите внимание: в зависимости от ширины формы часть команд может автоматически скрываться в меню Еще.
  • Записать и закрыть - записывает изменения и закрывает форму.
  • Записать - записывает изменения оставляя форму открытой.
  • Перечитать - отменяет все сделанные изменения и восстанавливает значения, сохраненные в информационной базе.
  • Скопировать - создает копию текущего элемента и открывает его форму для редактирования.
  • Пометить на удаление / снять пометку
    • - устанавливает или снимает пометку на удаление.
  • Изменить форму - открывает диалог для изменения внешнего вида формы. Можно изменить расположение полей и надписей и т.д.
  • Справка - открывает данный текст справки.
