Редактирование правил чтения файла

С помощью этого редактора можно настроить правило чтения практически любого файла. Единственное требование - файл должен конвертироваться в табличное представление. 

Для редактирования правила чтения нужно заполнить таблицу образца любым удобным способом:

  • открыть файл, содержащий данные образца
  • вставить табличное содержимое из буфера обмена

Справа от таблицы образца находится список показателей, которые могут быть заполнены из файла: кошелек, дата, сумма, статья и прочие реквизиты операций. Создание правил чтения заключается в указании ячеек или колонок, в которых программа должна искать значения показателей. Сделать это можно двумя способами:

1. Выделите в таблице справа нужный показатель, зажмите его левой кнопкой мыши и перетащите в нужную ячейку таблицы с образцом данных. Программа запишет адрес ячейки для выбранного показателя.
2. Вручную запишите адрес нужной ячейки в колонке "Положение в таблице"


3. Укажите, из каких ячеек таблицы программа должна извлекать значения показателей, необходимых для записи операций. 

Удобней всего это делать с помощью мыши: сначала перетащите мышью показатель "Начало таблицы операций" в первую ячейку таблицы, чтобы программа смогла отличать значения, расположенные в колонках таблицы от прочих показателей, например, расположенных в названии таблицы.
После этого перетаскивайте остальные показатели в колонку или ячейку, из которых нужно извлекать их значения.

Если какой-либо текст нужно составить из несколькиз ячеек и колонок, перетащите название показателя в нужные ячейки удерживая в нажатом положении клавишу Ctrl.
Сохраните правило и выполните чтение выписки.
Аналогчиным образом можно изменить созданные ранее правила чтения выписки.
