Справочник "Контакты"

Справочник содержит информацию о людях, организациях и предприятиях, отношения с которыми требуется отразить в учете.

Во первых, у пользователя с некоторыми контактами могут возникнуть долговые отношения: пользователь может выдать кому-то в долг деньги или свое имущество, или выполнить работы и ожидать оплату. Возможна и обратная ситуация, когда пользователь берет у кого-то деньги или имущество в кредит, или обязуется оплатить полученные услуги и работы. Такие отношения учитываются в программе с помощью долгов, с каждым контактом пользователь может может учитывать по нескольку долгов одновременно.

Во вторых, контакты могут быть использованы в дополнительной аналитике доходов и расходов пользователя. Например, при желании, учет доходов можно вести в разрезе заказчиков и работодателей, а учет расходов - в разрезе членов семьи и мест траты. Пользователь может сам настроить себе состав дополнительной аналитики, в которой могут быть использованы его контакты.

Для удобства, контакты можно объединить в группы.

Команды списка справочника

Для открытия списка справочника используйте команду Контакты в меню Справочники панели команд. Управлять содержимым справочника можно с помощью следующих команд:

Обратите внимание: в зависимости от ширины формы часть команд может автоматически скрываться в меню Еще.
  • Создать - открывает форму нового элемента справочника.
  • Создать группу  - открывает форму новой группы.
  • Скопировать - создает копию выделенного в списке элемента и открывает его форму для редактирования.
  • Изменить - открывает форму элемента для изменения.
  • Долги - открывает список долгов выделенного контакта.
  • Пометить на удаление / снять пометку
    • - устанавливает или снимает пометку на удаление.
  • Обновить - обновляет список справочника для отображения изменений, сделанных на другом компьютере или в другой сессии.
  • Режим просмотра - открывает меню для изменения режима просмотра списка - в виде дерева, в виде сплошного списка всех групп и элементов, или в виде иерархического списка, в котором содержимое каждой группы просматривается отдельно от прочих групп и элементов.
  • Переместить в группу - открывает диалог выбора группы, в которую следует переместить выделенные в списке записи.
  • Перейти на уровень выше - выделяет в списке группу, в которую входит текущий элемент. 
  • Перейти на уровень ниже - выделяет первый элемент, входящий в выделенную группу.
  • Найти - открывает диалог для поиска элементов справочника. При применении поиска в списке справочника останутся только те элементы, которые содержат искомую информацию.
  • Отменить поиск - отменяет отбор, наложенный на список командной Найти.
  • Настроить список - открывает диалог, в котором можно задать сортировку, отбор и условное оформление списка справочника.
  • Выбрать настройки - открывает диалог выбора настроек, сохраненных пользователем ранее.
  • Сохранить настройки - открывает диалог для сохранения настроек списка под каким-либо именем, для быстрого примерения в дальнейшем.
  • Установить стандартные настройки - сбрасывает все изменения настроек, сделанные пользователем.
  • Вывести список - открывает диалог для вывода списка в табличный документ (отчет).
  • Изменить форму - открывает диалог для изменения внешнего вида формы. Можно изменить расположение полей и надписей, изменить колонки и т.д.
  • Справка - открывает данный текст справки.
