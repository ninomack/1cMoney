Регистр сведений "Ключи статей"

В регистре "Ключи статей" содержится вспомогательная информация по элементам разных справочников, которая используется при построении иерархии статей в бюжете и при вычислении бюджетных показателей.

Записи в данный регистр добавляются автоматически при создании новых или изменении существующих элементов справочников. 
Изменение записей этого регистра вручную нежелательно.
В случае, если по каким-либо причинам иерархия статей в бюджете нарушится, для исправления ситуации можно обновить все ключи командой "Обновить ключи статей", которая доступна из формы списка.

Дополнительная информация: Бюджет
