Регистр сведений "Состав статей бюджета"

Регистр сведений "Состав статей бюджета" хранит информацию о пользовательских настройках статей бюджета для каждого варианта бюджета. Записи добавляются

Изменение записей этого регистра вручную нежелательно.
В случае, если по каким-либо причинам иерархия статей в бюджете нарушится, для исправления ситуации можно обновить все ключи командой "Обновить ключи статей", которая доступна из формы списка.

Дополнительная информация: Бюджет
